Objectives

Based on the insights gathered, a product should target following objectives


  • Saving time for the coders
  • Managing complexity by helping in organisation of code
  • Better experience by easy navigation through the code
  • Providing tools for an in-depth analysis of code logic
  • Increasing uptake of existing features

Conclusion


We saw improvements in Online IDE based testing environment targeting goals like time saving, code organisation, code navigation and in-depth analysis.
Some challenges in this field would be large set of first time users, a stressful testing environment and low discoverability of large number of features.
This would make experimenting with features indispensable for long term product success.
