Objectives

Based on the insights gathered, a product should target following objectives


  • Saving time for the coders
  • Managing complexity by helping in organisation of code
  • Better experience by easy navigation through the code
  • Providing tools for an in-depth analysis of code logic
  • Increasing uptake of existing features
